Rgrd

A R language package to make easy and fun play with Global Device Repositories

View on GitHub Download .zip Download .tar.gz

Welcome to Rgrd package.

This packages helps you to interact with a ereuse.org Global Devide Repository (GRD).

What this package does.

Below some examples you can run without any installation:

  • Get devices information from a GRD. Get GRD Devices Data

Rgrd Manual

Get introduced to this package using our Rgrd manual

How to manually download and install last version of this package

Download it as a .zip or .tar.gz, decompress it, install on R a file with name Rgrd_[version].tar.gz

How to install Rgrd package using devtools

On R type: (after devtools package installation)

library(devtools)<br> install_github("eReuse/Rgrd/Rgrd")<br>

How to install a test version of this package

On R type:

install.packages("http://data.ereuse.org/packages/Rgrd_0.01.tar.gz", repos = NULL, type="source")

How start with this package

On R type: (after installation on your R)

library(Rgrd)
url <- "http://sandbox.ereuse.org/api/devices/"
data <- getDevices(url)
result <- data$result
